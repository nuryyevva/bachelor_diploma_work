# bachelor_diploma_work
Разработка и сравнение адаптивных суррогатных моделей на основе гауссовских процессов для предсказания критической нагрузки на потерю устойчивости композитной панели. Модели обучаются на синтетических данных, а их адаптивность достигается за счёт активного выбора точек с целью минимизировать число итераций при заданной точности.
